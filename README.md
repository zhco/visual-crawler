# 可视化爬虫 - Via 浏览器用户脚本

在 Via 浏览器中安装此脚本，打开任意网页即可可视化爬取数据。

## 安装

1. 复制以下链接：
   `https://raw.githubusercontent.com/zhco/visual-crawler/main/%E5%8F%AF%E8%A7%86%E5%8C%96%E7%88%AC%E8%99%AB.user.js`

2. Via 浏览器 → 设置 → 脚本 → 右上角 `+` → 从 URL 导入 → 粘贴链接 → 确定

## 使用

| 步骤 | 操作 |
|------|------|
| 1 | 打开目标网页，从底部上滑展开面板 |
| 2 | 模式选「选列表项」→ 点击一个重复的卡片 |
| 3 | 切到「选字段」→ 逐一点击卡片内的标题/价格等 |
| 4 | 选好分页 → 点「开始爬取」→ 自动导出 CSV/JSON |

## 功能

- 可视化点击选取，自动生成 CSS 选择器
- 支持跨页爬取（fetch）  
- 导出 CSV（UTF-8 BOM）或 JSON
- 零依赖，纯浏览器端运行
